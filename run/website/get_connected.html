Instructions for Connecting

Hi! Thanks for your interest in EVIX. Anyone is welcome to join but please read the rules below to make everyone's lives easier.

As EVIX is a full internet exchange, you must have a non-reserved ASN assigned to you by an RIR. If you don't have an ASN yet. We recommend experimenting with DN42 until you are sure that you want to go the route of registering an ASN. Also, due to the limited amount of IPv4 that we have available. Please only request it if you are planning on announcing IPv4 space. You can always get an IPv4 address later if needed. We will check if you have IPv4 space before granting an IPv4 address but you can always let us know if you have a special situation. We are all here to learn so we are happy to accommodate.

Please note, only local connections are supported at our FRA and VAN POPs. If you wish to connect by a tunnel, please select a pop from the locations list. If you are connecting via a VM interface (such as ones offered by iFog or Free Range Cloud), please select the local connection type! I put this in bold because many people select the wrong connection type for a VM or request tunnels for a local-only location.

All that is left is for you to fill out the application! You won't receive a reply until we look at your application. Note that it may take a while before you get a reply. We are volunteer driven so you will just have to wait until we have time. This is especially important during the school year when many of our admins do not have the time to work on this project. I apologise if this causes any difficulties but this is just for fun so don't get too worked up about it. If you are rejected for any reason (only happened once) we will let you know so don't worry about that.

Please whitelist 'helpdesk@evix.org' in your mail client to avoid missing these important instructions. We have ended up on blacklists a few times.
